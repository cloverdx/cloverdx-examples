# Overview

Example project used during CloverDx 5.16 release webinar. Shows how to use libraries and how to work with HL7 data.

# Requirements
- DataTrackerLib installed on the Server. Development version from the webinar: https://bitbucket.org/cloveretl/partnerportal/src/DataTrackerLib-dev/DataTrackerLib/
- DataTracker web app installed on the Server. See [DataTrackerWeb](https://bitbucket.org/cloveretl/publicexamples/src/master/CloverDXReleaseWebinar-5.16/DataTrackerWeb/) for the details.
