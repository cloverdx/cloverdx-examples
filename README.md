# CloverDX Examples

This is the official GitHub repository containing examples that demonstrate the
usage of CloverDX. The repository contains several several projects showing
different features of CloverDX platform - starting with simple features showing
basic transformations all the way to more complex projects showing AI usage,
job orchestration, and more.

Each project includes README file which provides additional details about any
pre-requisites it may have (e.g., database, API key, ...). 

## Included Projects

| Project | Description |
|---------|-------------|
| [BasicFeatures](./BasicFeatures/) | A collection of simple examples showing variety of CloverDX features. |
| [DataAppExamples](./DataAppExamples/) | A collection of examples showing basic Data App use cases and extensions using JS. |
| [DataQualityExamples](./DataQualityExamples/) | Examples showing how to work with Data Quality components in CloverDX. |
| [IntroExamples](./IntroExamples/) | A collection of basic examples showing various CloverDX features. |
| [KafkaExamples](./KafkaExamples/) | A collection of examples showing integration with Apache Kafka. |
| [WranglerTutorial](./WranglerTutorial/) | Examples files for Wrangler tutorial from [CloverDX Documentation](https://doc.cloverdx.com/latest/user/wrangler-tutorial.html) |
